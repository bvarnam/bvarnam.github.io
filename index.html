Projects

Below are a few projects I have developed with a link to their respective github repos and a brief explanation.

Subreddit Categorization

This Natural Language Processing (NLP) project utilizes PushShift Reddit API to pull posts and comments from two related subreddits (ExplainLikeImFive and AskScience), then attempts to classify which subreddit the posts belong to through Vectorization and Machine Learning.

Kobe Shot Prediction

Originally from Kaggle, this project aims to take in game and shot data in order to predict whter or not Kobe Bryant will make his shot using multiple types of models and extensive exploratory analysis. The form of this project was a "Hackathon" done in the course of 2 days.

Stock Price Prediction (In Progress)

My first algorithmic attempt at beating the market, this project utilizes domain knowledge of day trading and Machine Learning in an attempt to predict the price of the SPY (S&P500 Index ETF) over the next 1-15 minutes.

Prison Sentencing Analysis and Prediction

A collaborative project looking at Prison Sentencing data and utilizing Machine Learning in order to determine if there is noticable discrimination in the sentencing portion of the criminal justice system, in regard to drug related crimes.
